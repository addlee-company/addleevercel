# Addlee-Vercel
Addlee company WebApp

## Deployment

This site is deployed on Vercel at: https://addleevercel.vercel.app

### Cache Control

The `vercel.json` configuration file ensures that:
- Static files are served with proper cache headers
- Browser caching is prevented with meta tags in HTML
- Changes deployed to the main branch are immediately visible

### Files

- `index.html` - Main landing page with "Welcome to Addlee"
- `addlee_website.html` - Alternative version with navigation (About, Waitlist, Contact)
- `vercel.json` - Vercel deployment configuration
